# Recursos de Investigación del Proyecto

## 1. Contexto del Documento

Este documento consolida las referencias académicas y técnicas que
fundamentan la investigación del proyecto. Su propósito es centralizar
las fuentes utilizadas en el análisis conceptual y técnico sobre:

-   Sistemas conversacionales multi-intención.
-   Arquitecturas multi-agente.
-   Implementaciones con LangGraph y LangChain.
-   Diseño de aplicaciones conversacionales multilingües.

Las referencias están formateadas bajo el estándar **APA (7ª edición)**.
